                   HARMONIA DAS CORES

O circulo cromático é importantíssimo para a harmonia de cores!

Temos as cores primárias,secundárias e terceirárias

Ao fazer um triângulo equilátero no circulo, veremos as cores primárias

CORES TERCEIRÁRIAS: ao usarmos uma cor terciária, nomearemos-a com o nome da cor primária e o nome da cor secundária.

CORES COMPLEMENTARES: para encontrarmos as cores complementares, usaremos a que esta do outro lado do circulo cromático

CORES ANÁLOGAS: cores análogas são cores em que não existe um contraste tão grande entre elas, porém, são perceptíveis. Para isso, pegaremos as cores ao lado

CORES ANÁLOGAS RELACIONADAS: pegaremos somente UMA cor vizinha, e logo após, pegaremos outra cor ao lado pulando uma.

CORES INTERCALADAS: escolheremos uma cor central, logo após, pularemos uma cor, e pegaremos a cor seguinte, e entao, faremos isso mais uma vez. No final, teremos uma paleta com 3 cores.

CORES TRIÁDICAS: É praticamente mesma coisa das cores primárias, porem, o triangulo equilatero pode ser movido para qualquer direção.

CORES EM QUADRADO: pegamos uma cor, logo depois pulamos 2 cores e pegamos esta cor. faremos isso 3 vezes, para no final obter 4 cores.

CORES TETRÁDICAS: escolhemos 1 cor, pegamos a do lado oposto desta cor. Faremos isso 2 vezes, para assim obtermos 4 cores.

Podemos criar uma paleta usando Cores Análogas e uma complementar. Por exemplo, as duas cores vizinhas do roxo, e uma complementar, no caso, a que esta do outro lado do circulo cromatico

                          SELETORES

Para podermos modificar somente um elemento dentre varios, colocamos um identificador nele. Podemos ter vários h1 em um mesmo código, porém, se fizermos <h1 id="principal">, estamos atribuindo um identificador nele, que na hora de modificar com css, faremos h1#nome do id

Em HTML é id, em css é #
Em HTML é class, em css é .

Representamos pseudoclasses com dois pointos(:), pseudoelemtos com dois pontos duas vezes(::), e por fim, > significa filho, ou seja, que uma tag esta dentro de outra.

Uma classe funciona da seguinte maneira: vamos declarar o seu nome, e apos isso, colocamos os estilos que irao conter nela. Com uma classe "texto", por exemplo, todas as tags que tiverem class="texto", irão ser modificadas de acordo com o que tiver dentro da classe texto nas css. .texto{text-align:center;}

a tag <span> por si só nao serve pra nada, porem, quando colocamos uma classe nela, o conteudo dentro dessa tag mudara de acordo com o que esta sendo especificado na classe.

Podemos usar mais de uma classe em determinada tag, basta apenas especificar o nome delas, por exemplo, class="basico principal".

border: serve para colocar uma borda, como o próprio nome já sugere. border: tamano, tipo, cor.

display: inline-block; deixa tudo na mesma linha.



  Pseudo-Classes:

tag:hover{} significa que quando voce passar o mouse por cima, irá acontecer o que esta dentro das chaves.

Para esconder um texto, podemos fazer igual a este exemplo: div > p{display none}; esse codigo se traduz basicamente "todos os paragravos que estiverem dentro de div, não serão mostrados".

Pseudo classe hover: voce escolhe o que acontecerá quando passar o mouse por cima do elemento.

active: determina como o elemento vai ficar quando voce segurar com o botao do mouse em cima dele.

visited: permite que voce escolha a maneira que o elemento vai ficar apos ser visitado.

  Pseudo-Elementos:

after: pode determinar o que vai aparecer no final do conteúdo.Por exemplo, no final de todos os parágrafos, colocamos o texto "paragrafo".

before: determina o que vem antes do conteudo.


                               MODELO DE CAIXAS

padding é o preenchimento de uma caixa. Serve para a borda fique distante do conteudo.

margin serve para dar um espaço externo.

outline serve para dar um traçado na borda do elemento.

TIPOS DE CAIXA:

box-level: ocupa 100% da largura da viewport e gera uma quebra de linha

inline-level: só ocupa a largura do elemento mostrado e nao gera quebra de linha.

tanto em margin como em padding, podemos espeficiar a largura inferior, do topo, da esquerda e da direita.

user agent significa que é o navegador que esta exibindo o conteudo.

Ao inspecionar uma página, podemos ir em estilos e alterar os valores sem modificar o arquivo.

Para toda caixa que desejamos que fique no meio da página, usamos margin:auto;

Podemos usar shorthands para simplificar as propiedades da bord, outline, padding e margin.

shorthand bord: width, style, color.

shorthand padding: top, right, bottom, left.

shorthand margin: top, right, bottom, left.

shortdhand outline: width, style, color.
